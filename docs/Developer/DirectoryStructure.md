# ディレクトリ構造

```txt
├─.github/
│  └─workflows/           自動テストの設定ファイル置き場
├─docs/                   ドキュメント
├─example/                サンプルコード
├─src/                    本ライブラリのインクルードディレクトリ
│  └─Udon/              　本ライブラリのファイルがある場所
├─test/                 　自動テスト用ソースファイル
├─.clang-format         　ファイルの自動整形用設定ファイル
├─.gitignore            　gitでコード管理したくないファイルやディレクトリを登録しておくファイル
├─.gitmodules             本リポジトリに追加されている外部のリポジトリがまとめられたファイル
├─library.properties      本ライブラリの各種情報をArduino IDEに提供するためのファイル
├─LISENCE                 ライセンス
└─README.md               GitHub当で表示されるメインページ
```
